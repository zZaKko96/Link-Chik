### 📝 Завдання
Адаптувати програмний код ЛР№2 до вимог фреймворка VueJS та забезпечити збереження і завантаження даних за допомогою власного Web-сервера та бази даних.

**Реалізований функціонал:**
* **Frontend (Vue.js 3):** Інтерфейс повністю перенесено на Vue. Використано реактивність для зв'язку даних із формою та таблицями. Взаємодія з бекендом реалізована через асинхронні запити `fetch` (REST API).
* **Backend (Node.js + Express):** Створено сервер для обробки запитів на реєстрацію, вхід, створення, видалення та редагування посилань.
* **База даних (SQLite):** Дані користувачів та посилань зберігаються у реляційній базі даних на сервері.
* **Функції:** Повноцінна автентифікація, лічильник кліків, керування профілем та генерація коротких посилань через зовнішній API.

### 🛠️ Технології
* **Vue.js 3** (Frontend фреймворк)
* **Node.js & Express.js** (Backend сервер)
* **SQLite** (База даних)
* **Bootstrap 5** (Стилізація та адаптивність)
