### 📝 Завдання
Розробити функціональність для статичних сторінок Web-додатку із використанням архітектурного шаблону **MVC (Model-View-Controller)**. 

**Реалізований функціонал:**
* **Автентифікація:** Реєстрація, вхід (з функцією «Запам'ятати мене» через `localStorage`/`sessionStorage`) та вихід з акаунту.
* **Профіль користувача:** Перегляд статистики (кліки, кількість посилань), редагування особистих даних, зміна фото та видалення профілю.
* **Керування посиланнями:** Створення коротких URL через інтеграцію з API, перегляд списку, редагування оригінальних адрес та видалення записів.
* **Локальне сховище:** Усі дані користувачів та посилання зберігаються локально в браузері (структури даних масивів та об'єктів у `localStorage`), що імітує роботу повноцінної бази даних.

### 🛠️ Технології та Патерни
* **JavaScript (ES6+)** — основна мова програмування логіки
* **MVC (Model, View, Controller)** — патерн організації коду (модульна структура)
* **HTML5 & CSS3** — структура та стилізація
* **Bootstrap 5** — UI фреймворк для забезпечення адаптивності
* **GitHub Pages** — розгортання додатку
