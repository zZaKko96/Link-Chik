### 📝 Завдання
Розробити статичний Web-інтерфейс для додатку скорочення посилань. Додаток має надавати можливість:
* Ведення списку URL користувача.
* Додавання, перегляду, вилучення та редагування посилань.
* Генерації скороченого вигляду URL.
* Адаптивного відображення на різних пристроях (мобільний, планшет, ПК).

### 🛠️ Технології
* **HTML5** (структура)
* **CSS3** (стилізація)
* **Bootstrap 5** (фреймворк для адаптивності та компонентів)
* **GitHub Pages** (хостинг)
